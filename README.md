# Olly Henson Coaching — Homepage

Custom-coded homepage for ollyhenson.com, built in GHL Website Builder.

## Quick Start
1. Open GHL Website Builder > ollyhenson.com
2. Add a Full Width section with a Custom HTML block
3. Set section + column padding/margin to 0
4. Paste contents of `sections/homepage.html`
5. Save and preview
